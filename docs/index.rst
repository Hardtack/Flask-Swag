Flask-Swag provides utilities for extracting `Swagger <http://swagger.io/>`_
spec from Flask application.

With Flask-Swag, you can easily generate an API document from your flask application.

It provides:

*   Core utilities to generate & validate Swagger spec.

*   Swagger spec extractor for flask view functions.

*   Swagger UI blueprint & Flask extension.
